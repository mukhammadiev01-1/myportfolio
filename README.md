# Shokhjakhon Mukhammadiev — Portfolio

A futuristic engineering portfolio built with Next.js, React and TypeScript.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
npm start
```

## Current sections

- Engineering-focused hero experience
- Selected systems: KidsGarden, Royal Food and PlantCare AI
- Full-stack architecture map
- Interactive command terminal
- Responsive layout for desktop and mobile

Contact details, screenshots and project case studies will be refined as additional portfolio materials are added.
